## Objetivo do projeto

Esta página foi construída com foco em:

✅ Conversão de tráfego pago (Meta Ads / Google Ads)  
✅ Presell page para afiliado  
✅ Aumentar CTR para marketplace  
✅ Melhorar qualificação do clique  
✅ Capturar eventos analíticos (GA4 / Pixel)  
✅ Carregamento rápido  
✅ SEO on-page  
✅ Excelente UX em dispositivos móveis

## Configuração

Abra o arquivo:

```js
const affiliateLink = "COLOCAR_LINK_AQUI";
```

Substitua pelo seu link de afiliado.

Exemplo:

```js
const affiliateLink = "https://seulink.com";
```

Todos os botões usarão automaticamente esse link.
